## Overview

The Tripedge TypeScript SDK provides a strongly-typed client for interacting with the Tripedge API. It includes methods for managing users, bookings, and accounting entries.

## Installation

To install the Tripedge TypeScript SDK, you can use either npm or yarn:

### Using npm

```bash
npm install tripedge-sdk
```

### Using yarn

```bash
yarn add tripedge-sdk
```

## Getting Started

After installing the SDK, you can import and initialize it in your project:

```typescript
import { TripedgeBackendAPI, CreateUserPayload } from 'tripedge-sdk';

// Initialize the API client
const api = new TripedgeBackendAPI();

// Example: Creating a new user
const createUserPayload: CreateUserPayload = {
  firstName: 'John',
  lastName: 'Doe',
  email: 'john.doe@example.com',
  phone: '1234567890',
  clientAccountLimit: 1000,
};

api.user.createUser(createUserPayload)
  .then(response => {
    if (response.success && response.data) {
      console.log('User created:', response.data);
    } else {
      console.error('Error creating user:', response.message);
    }
  })
  .catch(error => {
    console.error('API error:', error);
  });
```

  ```typescript
    firstName: 'Jane',

  };

  api.user.createUser(newUser)
    .then(response => {
      // Handle response
    });
  ```

- **Get User by ID**

  ```typescript
  const userId = '123';

  api.user.getUserById(userId)
    .then(response => {
      // Handle response
    });
  ```

